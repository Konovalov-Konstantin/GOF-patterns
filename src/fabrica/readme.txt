
                                            Паттерн ФАБРИКА.

Паттерн Фабрика инкапсулирует создание объектов в зависимости от входящего параметра. Фабрика также может быть непараметризованной
и производить только один вид объектов.
Есть 2 фабрики (ChicagoPizzaStore, NYPizzaStore), унаследованные от абстрактного класса PizzaStore. Эти фабрики релизуют
абстрактный метод createPizza и создают конкретный объект типа Pizza в зависимостиот переданного параметра type.
createPizza вызывается в orderPizza, в котором prepare, bake, cut и box реализованы по дефолту в абстрактном классе Pizza.
Метод cut переопределен в классе ChicagoStyleCheesePizza.

PizzaStore - абстрактная фабрика (ChicagoPizzaStore, NYPizzaStore - конкретные реализации фабрик)
PizzaStore.createPizza - фабричный метод
